克劳修斯熵
玻尔兹曼熵：klnp，p越大对应系统混乱越大(微观状态数最多) => 最有可能的状态
信息熵：lnp => 事件p发生的概率越大，它所能携带的信息就越少
信息的价值在于消除不确定性
所谓信息量 => 不确定性减少的程度
使用log的本质在于对于A事件和B事件同时发生，信息量是相加，对应的概率是是相乘
而所谓log以谁为底是由进制决定的 => 二进制 => log以2为底
N是指一个事件等可能的数量，因为任何概率都可以翻译为等可能事件 => 1 / N
转换为摸球问题
计算信息和的时候，我们是log算出具体的值，但两个事件发生概率也不同，所以还要分别乘以对应的概率
意义：底数进制的最大极限

原因在于：物理熵指的是系统最终最稳定的状态(越集中越固定，熵越小；越分散越难以预测唯一，熵越大)
         信息熵描述的是事件携带的信息量 => 越不确定越大

/ ln n是为了归一化

计算指标的混乱程度
