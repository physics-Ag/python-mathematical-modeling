层次分析法原理：

一、基本概念补充：
1、一致性矩阵：满足a(i, j) * a(j, k) = a(i, k)的矩阵

2、正互反矩阵：满足a(i, i) = 1，a(i, j) = 1 / a(j, i), a(i, j) > 0

3、一致性矩阵和正互反矩阵的区别：一致性矩阵在所有元素都大于的0的条件下就是正互反矩阵
                              证明：令i = j = k => a(i, i) * a(i, i) = a(i, i) => a(i, i) = 1
                                    令i = k => a(i, i) = a(i, j) * a(j, i) = 1

4、任何元素都大于0的n阶一致性矩阵其特征值一定有一个为n，另外(n - 1)个为0
            证明：由于矩阵的n个特征值之和为n 且 n个特征值之积为矩阵的行列式(易知矩阵的行列式一定为0) => 结论

5、任何非一致的n阶正互反矩阵A，其最大特征值 λ_max(A) > n

二、层次分析法使用场景
我们要分析多个因素对一个事物的影响 => 这个时候就需要对这些因素的权重进行分析 => 此时就可以使用层次分析法了

三、原理
(1)、我们将要分析的n个因素写成n * n的矩阵a的形式，a(i, j)表示i相对于j的重要程度，大于1表示i比j重要(越大表示越重要)，小于1表示j比i重要(越小表示越重要)
(2)、对于该矩阵我们需要验证其一致性是否在一个合理的限度内
    解释：所谓一致性抛开数学定义，就是i比j大2倍，j比k大2倍 => i比k大4倍 => 其实你仔细看这段话，他不就是一致性矩阵的定义嘛
(3)、而衡量这个与一致性的偏差，我们用最大特征值λ_max(A)与n的差值来衡量
    解释：如果就是一致性矩阵，那结果最大特征值就是n，但如果不是一致性矩阵，由上面的概念补充，它就一定会大于n，并且越不像一致性矩阵，最大特征值就越大！
    (记住即可，不予以证明)
(4)、于是可以得到一致性指标CI = (λ_max(A) - n) / (n - 1)
    解释：如果我们想要用一个统一的指标验证其一致性是否在一个合理的限度内，仅仅依靠λ_max(A) - n还不够，因为对于不同阶的矩阵不公平，所以这里除以(n - 1)
(5)、我们再来计算RI(随机生成一组数据，计算其CI)，一致性就用CI / RI < 0.1
    解释：由于不同阶数的矩阵的CI的“可接受阈值”并不相同，也就是说n越大，即使随机乱填矩阵，得到的CI也会越大
